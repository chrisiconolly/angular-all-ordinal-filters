Anular All Ordinal Demo

Angular All Ordinal provides you with 2 angular filters which add ordinals to numbers. View the project on GitHub

Usage

The two filters provided are "ordinal" and "ordinalOnly", you can see their usage below.

{{number | ordinal}} // 1 -> 1st
{{number | ordinalOnly}} // 1 -> st

Example

Number Ordinal-only Ordinal
{{number}} {{number | ordinalOnly}} {{number | ordinal}}
